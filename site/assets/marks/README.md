# Innsigle seal marks (exploration pack)

Provisional SVG pack for DESIGN.md state cues. Uses `currentColor` so hosts
set ink via CSS. One glyph family; letter cues **H** / **M** / **A**.

| File | Composition |
|------|-------------|
| `innsigle-base.svg` | No state letter (tiny footer) |
| `innsigle-human.svg` | `human-authored` |
| `innsigle-mixed.svg` | `mixed` |
| `innsigle-model.svg` | `model-primary` |

## Do / don't

| Do | Don't |
|----|--------|
| Monochrome / high contrast | Green = human, red = AI |
| Link mark to mash bill | Imply "content is true" |
| Use same family for all states | Separate purity brand for human only |

## Export sizes

For social: render PNG at 64, 128, 256. Footer: inline SVG at 24–32px height.
