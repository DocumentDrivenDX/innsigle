Mash bill

The mash bill is the recipe of making—not a moral grade. Like a distiller’s grain bill: ingredients and roles, not “good” vs “bad.”

Composition

ValueWhenMark cue
human-authored Humans produced the substantive work; models not primary authors H
mixed Material human and model authorship both present M
model-primary Model(s) produced most substantive content; humans may edit A

Ingredients

Each line is a tool, model, or human role:

FieldRule
kindmodel · tool · human · other
namee.g. Claude, sloptimizer, operator
rolee.g. draft, rewrite, structure-edit
version / uriOptional

Edit is not origin

Running sloptimizer (or similar) on model output does not flip model-primary to human-authored. List the tool as kind: tool. Sounding less “AI” is prose quality, not composition.

Example (docs)

{
  "schema_version": "1",
  "composition": "model-primary",
  "ingredients": [
    { "kind": "model", "name": "Claude", "role": "draft" },
    { "kind": "tool", "name": "sloptimizer", "role": "rewrite" },
    { "kind": "human", "name": "operator", "role": "structure-edit" }
  ]
}
