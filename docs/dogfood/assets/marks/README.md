# Innsigle seal marks

Solid **stamp disks** for DESIGN.md composition cues. Opaque fill so marks
read as `<img>` assets (not wireframes on transparent backgrounds).

Ink: near-black disk `#1c1916` · light engravings `#e8e2d6`. Works on light and
dark page paper; the bright rim keeps the disk visible on dark themes.

| File | Composition | Cue |
|------|-------------|-----|
| `innsigle-base.svg` | Base (no state letter) | — |
| `innsigle-human.svg` | `human-authored` | H badge |
| `innsigle-mixed.svg` | `mixed` | M badge |
| `innsigle-model.svg` | `model-primary` | A badge |

## Do / don't

| Do | Don't |
|----|--------|
| Solid high-contrast disk | Green = human, red = AI |
| Link mark to colophon | Imply "content is true" |
| Same family for all states | Separate purity brand for human only |
| Footer ≥40px height | Hairline wireframe at 16px only |
