# Innsigle seal marks

Solid **stamp-matrix** faces. Inspired by ancient stamp seals, medieval merchant
marks / guild touches, pottery cartouches, and hallmark countermarks — not by
UI wireframes. See `RESEARCH.md`.

**Ink:** matrix `#141210` · impression `#e4dccf`.

| File | Composition | Countermark |

## Anatomy

```text
  ┌── beaded outer rim ──┐
  │   inner field ring    │
  │    monogram (stave    │
  │     + thryst +        │
  │      lozenge)         │
  │   [optional punch]    │  ← hallmark-style composition cue
  └───────────────────────┘
```

## Do / don't

| Do | Don't |
|----|--------|
| High-contrast solid matrix | Green = human, red = AI |
| Link mark to colophon | Imply "content is true" |
| Same family for all states | Separate purity brand for human only |
| Angular proprietary monogram | Copied historical house marks or neopagan symbols |
