## Name decision: Innsigle

| Field | Value |
|-------|--------|
| **Public brand** | **Innsigle** |
| **Etymon** | Old Norse / Icelandic **innsigli** — seal, seal-ring; wax on a deed |
| **Spelling choice** | **Innsigle** (anglicized product form). Historical lemma is *innsigli* (double *n*, ends in *i*). Brand spelling favors English word-shape while keeping the seal lineage. |
| **Pronunciation (locked default)** | **INN-siggle** — rhymes with *single*; stress on **INN**. One word, two syllables after “Inn”: *Inn-siggle*. |
| **Why this name** | True document-seal word family; Norse/Azgaard fit; avoids bare **Sigli** (live AI software company at sigli.com); visual seal carries recognition so the string need not be instantly English |
| **Not chosen** | Sigli (collision), Segl (ON = sail), AI Badge, Mashbill-as-name, English Signet-alone |

### Brand architecture

| Layer | Role | Name |
|-------|------|------|
| House | Studio / operator trust root | **Azgaard** |
| Product | Public seal system | **Innsigle** |
| Concept words | Explainer vocabulary | signet, colophon, seal, claim |
| Visual primary | Mark readers recognize | **Innsigle seal mark** (design system TBD) |
| Repo / code | Engineering slug | `aibadge` until rename |

Ship pattern: *“Innsigle by Azgaard”* or seal.azgaard.net / innsigle.* when domains settle.

### Name + visual

Operator note: **the key element is a visual branding element.** Wordmark and spoken name support the **seal mark**; recognition lives first in the glyph/system, second in the string. Naming doc does not lock mark design—that is design-system / brand work after frame or in parallel discovery.

### Guardrails

- Do **not** market as AI detection.
- Do **not** use “Credentials” (C2PA collision).
- Do **not** ship public brand as **Sigli** alone.
- Always separate **declared colophon** and **signed seal** from **detected fact**.
- Etymology honesty: Latin *sigillum* line via North Germanic legal vocabulary—not a pre-Christian pure root. Say “Norse/Icelandic for a document seal,” not “ancient pagan word for truth.”

No other variants. Brand audio/video uses this once; the **mark** carries recognition.
