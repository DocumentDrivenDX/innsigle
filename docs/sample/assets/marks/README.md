# Innsigle seal marks

Choosable **stamp families** × composition **countermarks**. Solid matrices
(not transparent wireframes). Research: `RESEARCH.md`.

**Ink:** matrix `#141210` · impression `#e4dccf`.

## Families

| Family | Files | Voice |
|--------|-------|-------|
| **Matrix** (default site chrome) | `innsigle-*.svg` or `matrix-*.svg` | Beaded rim, merchant monogram |
| **Brand** | `brand-*.svg` | Heavy iron-brand silhouette |
| **Cartouche** | `cartouche-*.svg` | Pottery square plate in round stamp |
| **Ring** | `ring-*.svg` | Signet / concentric bezel |

## Composition suffixes

| Suffix | Composition |
|--------|-------------|
| `-base` | No cue (neutral chrome) |
| `-human` | `human-authored` (H) |
| `-mixed` | `mixed` (M) |
| `-model` | `model-primary` (A) |

Example: `brand-model.svg` = brand family + model-primary countermark.

## Do / don't

| Do | Don't |
|----|--------|
| High-contrast solid matrix | Green = human, red = AI |
| Link mark to colophon | Imply "content is true" |
| Same family across H/M/A | Separate purity brand for human only |
| Let operators pick a family | Copy real historic house marks or neopagan symbols |
